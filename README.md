# 📚 Stacks & Queues in Python

Implementation of the **Stack** and **Queue** abstract data types in Python, developed as part of the **Data Structures** course at the National University of Hurlingham (UNAHUR).

The project simulates the operation of a roadside assistance company in the Buenos Aires Metropolitan Area (AMBA), managing assistance requests, dispatch offices, and vehicle services through custom implementations of stacks and queues.

The main objective was to understand and implement fundamental data structures from scratch without relying on Python's built-in collections.

---

# 📚 Overview

The system models a roadside assistance company where customers request towing or repair services.

The implementation includes:

- Stack (LIFO) data structure
- Queue (FIFO) data structure
- Roadside assistance requests
- Dispatch offices
- Company building simulation
- Priority management
- Vehicle assignment

A testing script and CSV dataset were provided by the course instructors to validate the implementation.
